## 执行方法

``` bash
# install dependencies   安装依赖
npm install

# serve with hot reload at localhost:8081   启动项目
npm run dev
cd 到json文件中 npm run test
# build for production with minification    打包项目
npm run build

```

## 描述
根据 jsonserver搭建本地服务器添加数据，使用vue进行本地服务器数据增删改查
